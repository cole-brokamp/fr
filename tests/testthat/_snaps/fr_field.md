# fr_field works

    Code
      fr_field(x = 1, name = "x")
    Output
      <fr_field[1]>
      x (fr_number)
      [1] 1

---

    Code
      fr_field(factor(letters), name = "letters")
    Output
      <fr_field[26]>
      letters (fr_string)*
       [1] a b c d e f g h i j k l m n o p q r s t u v w x y z
